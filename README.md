\# 🖱️ AI Virtual Mouse



Control your system using hand gestures with computer vision.



\## 🚀 Features

\- Cursor movement using hand tracking

\- Click actions using gestures

\- Real-time performance



\## 🛠️ Tech Stack

\- Python

\- OpenCV

\- MediaPipe

\- NumPy

\- Autopy



\## ▶️ Run

pip install -r requirements.txt

python virtualtouch.py
